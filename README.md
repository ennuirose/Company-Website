# Company Website (Next.js + Tailwind CSS)

This is a modern **company website** project built with **Next.js 14** and **Tailwind CSS**, created as part of my portfolio to showcase full-stack web development skills.
The design focuses on clean UI, responsiveness, and real-world sections that most businesses need.

2

---

##  Features

- **Responsive design** (mobile, tablet, desktop ready)
- **Hero section** with background image & call-to-action
- **About page** introducing the company
- **Services page** listing offerings in a professional layout
- **Contact page** with a working form (ready to connect to a backend or service like Firebase / Email API)
- **Reusable components** (Navbar, Footer, Buttons, Layouts)
- Built with **Next.js App Router** and **TypeScript**

---

##  Tech Stack

- **Next.js 14** — React framework for production-ready apps
- **Tailwind CSS** — Utility-first CSS for rapid UI building
- **TypeScript** — Type safety for maintainable code
- **Vercel** — Deployment and hosting
